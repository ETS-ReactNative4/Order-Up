


## Order Up

### About Order Up
Oder Up is food ordering application written in react native.


### Usage

#### Download

```bash
	$ git clone git remote add origin https://github.com/Thagana/travSIM-Food.git
```

#### Install dependency

```bash
	$ npm install
```

#### Install expo
```bash
	$ npm install expo-cli -g
```

#### Run the application
```bash
	$ expo start -a
```

```bash
	$ expo start -i
```

### Build

```bash
	$ expo build:android
```

```bash
	$ ecpo build:ios
```
